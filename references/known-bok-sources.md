# Known Bodies of Knowledge and Expert-Curated Sources

When generating a roadmap, search for the relevant BoK or curated source from this list first.
This is not exhaustive — always search for domain-specific BoKs beyond this list.

## Computer Science / Software Engineering
- **ACM/IEEE CS2023** — 17 knowledge areas, core vs elective distinction
- **SWEBOK v4** — 15 knowledge areas for software engineering specifically
- **roadmap.sh** — Community-curated developer roadmaps (backend, frontend, DevOps, etc.)
- GitHub `awesome-*` lists — Community-curated tool/resource lists

## Project / Product Management
- **PMBOK (7th ed)** — Performance domains, principle-driven
- **Scrum Guide** — Official scrum framework
- **Shape Up (Basecamp)** — Alternative to scrum/agile

## Data
- **DAMA-DMBOK** — 11 functional areas of data management
- **CDMP certification path** — Data management professional certification

## Finance
- **CFA Curriculum** — 3-level structure with Learning Outcome Statements (LOS)

## Cloud / Infrastructure
- **AWS Certification Paths** — Role-based (Architect, Developer, Operations)
- **GCP Certification Paths** — Similar structure
- **Azure Certification Paths** — Similar structure
- **CNCF Landscape** — Cloud-native technology landscape

## Security
- **CISSP CBK** — 8 domains of information security
- **OWASP** — Web application security knowledge base

## Machine Learning / AI
- **Stanford CS229/CS231n syllabi** — ML and deep learning course structures
- **fast.ai curriculum** — Practical deep learning curriculum
- **Papers With Code** — State-of-art tracking with benchmarks

## Design / UX
- **Nielsen Norman Group articles** — UX research and principles
- **Design Systems** — Various company design system documentation

## Cross-Domain
- **ESCO** — European Skills, Competences, Qualifications and Occupations taxonomy
- **O*NET** — US occupational information network
- **Wikipedia category hierarchies** — Useful for discovering domain structure
- **University course catalogs** — Search "[domain] course sequence [top university]"
